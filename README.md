# CSV to JSON Convertor
A simple csv to json convertor

![Preview](./preview.png)

## Usage
-   Clone this repository
-   Change Executable Permission for *readlines* script file `chmod u+x ./readlines`
-   To run code:
    ```js
    node index /path/to/csv-file.csv
    ```
